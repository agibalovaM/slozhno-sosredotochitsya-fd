# 🧠 Проект "Сложно сосредоточиться"

## 📖 Описание:
Веб-страница, посвящённая теме концентрации внимания и причин её снижения.  
Проект выполнен в рамках обучения современным практикам вёрстки: от семантической структуры HTML до адаптивного оформления и работы с темами.  
Благодаря работе над этим проектом я глубже поняла, как построить масштабируемую систему стилей и реализовать переключение оформления с учётом системных настроек пользователя.

## 🚀 Реализовано:
- Семантическая HTML-структура  
- Переключение темы (день / неон / авто) с запоминанием состояния  
- Использование CSS custom properties (`:root`) и цветовых переменных  
- Адаптивная сетка с CSS Grid  
- Галерея с кроссбраузерной вёрсткой и lazy loading изображений  
- Модульная структура файлов  
- Поддержка `prefers-color-scheme` и media-запросов  
- Красиво оформленные интерактивные элементы (`:hover`, `:focus`, `:focus-visible`)  

## 🛠 Технологии:
- HTML5  
- CSS3 (Grid, переменные, media queries)  
- JS (переключение тем оформления)  
- Методология БЭМ  
- Адаптивная типографика и дизайн  

## 📦 Как использовать:
1. Клонируйте репозиторий  
2. Откройте `index.html` в браузере  

**Системные требования:**  
Современный браузер с поддержкой CSS Grid и `prefers-color-scheme` (Chrome, Firefox, Safari)
